# 浏览器缓存清理指南

## 问题描述

如果您遇到以下问题：
- 技能激活选项只显示江苏省的城市
- 界面显示异常
- 功能不正常更新

很可能是**浏览器缓存**导致的旧代码还在运行。请按以下步骤清理缓存。

---

## 解决方法

### 方法1：硬刷新（推荐，最快）

**Windows/Linux**:
- Chrome/Edge: `Ctrl + Shift + R` 或 `Ctrl + F5`
- Firefox: `Ctrl + Shift + R` 或 `Ctrl + F5`

**macOS**:
- Chrome/Edge: `Cmd + Shift + R`
- Firefox: `Cmd + Shift + R`
- Safari: `Cmd + Option + R`

---

### 方法2：清空缓存并硬刷新

#### Chrome/Edge:
1. 按 `F12` 打开开发者工具
2. **右键点击**浏览器地址栏旁的刷新按钮
3. 选择 **"清空缓存并硬性重新加载"**

#### Firefox:
1. 按 `Ctrl + Shift + Delete` (Windows/Linux) 或 `Cmd + Shift + Delete` (macOS)
2. 时间范围选择 **"全部"**
3. 勾选 **"缓存"**
4. 点击 **"立即清除"**
5. 刷新页面

#### Safari:
1. 菜单栏 → 开发 → 清空缓存
2. 或 `Cmd + Option + E`
3. 刷新页面

---

### 方法3：清空所有站点数据（终极方案）

#### Chrome/Edge:
1. 设置 → 隐私和安全 → 清除浏览数据
2. 时间范围：**全部时间**
3. 勾选：
   - ✅ Cookie 和其他网站数据
   - ✅ 缓存的图片和文件
4. 点击 **"清除数据"**

---

## 开发服务器重启（如果您在本地开发）

如果清理缓存后问题仍存在，请重启开发服务器：

```bash
# 停止当前运行的服务器 (Ctrl+C)
# 然后重新启动
npm run dev
```

---

## 验证是否成功

清理缓存后，请检查：

1. **技能激活选项**：
   - 选择浙江省的城市（如杭州市、宁波市）
   - 选择山东省的城市（如济南市、青岛市）
   - 应该都能看到"激活技能"复选框

2. **技能类型**：
   - 主动技能（`ACTIVE`）会显示"激活技能"选项
   - 被动技能（`PASSIVE`）不需要激活，自动触发

3. **技能日志**：
   - 激活技能并确认部署后
   - 打开浏览器控制台（F12 → Console）
   - 应该看到 `[城市技能]` 开头的调试日志
   - 战斗日志中应该显示技能效果

## 仍然有问题？

如果清理缓存后问题仍然存在，请：

1. 打开浏览器控制台（F12）
2. 切换到 **Console** 标签
3. 重现问题
4. 截图控制台输出（特别是红色错误信息）
5. 联系开发者并附上截图
